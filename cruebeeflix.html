Check out the movie app! | GitHub | API

Objective

Explore React by creating the server-side and client-side for a single page application with multiple user views.

Description

This app is all about movies, the directors behind them, and some additional information about the movie, director, and the genre of that movie. It features routing, complex user interaction, and several interfaceable views.
