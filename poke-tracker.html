Pokedex

Check out the Pokedex app! | GitHub

Objective

Explore advanced JavaScript concepts. Introduction to jQuery, AJAX, and API interaction. Grasp importance of good documentation and methods for reading and understanding documentation.

Description

The Pokedex app is a simple JavaScript application that interacts with an external API. The application pulls information from the API and displays the information based on user input. Users can search for specific characters, and display additional details about them with a click of a button. The application is built with HTML, CSS, JavaScript, jQuery, and Bootstrap.

  • Bootstrap
